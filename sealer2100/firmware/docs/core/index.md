# HyperMate Core

HyperMate Core is the second-gen firmware running on HyperMate devices. It currently runs on HyperMate Touch only, but it will probably be used on HyperMate mini/classic in future as well.

HyperMate Core is part of the HyperMate-firmware monorepo to be found on [GitHub](https://github.com/ziancube/hypermate-touch-firmware), in the `core` subdirectory.
